# 第三问第3小问：兼顾经济性、电池寿命与碳排放的综合策略

## 1. 分析目标

在三个维度（经济性 / 电池寿命 / 碳排放）之间取得合理平衡，形成**定量化**的综合运行策略。不能只停留在"用 S5 统一优化"的定性描述。

## 2. 总体计算流程

```
步骤1：计算碳减排潜力上限（最小可行碳排放）       ← 基准锚点
步骤2：cap × price 联合场景分析（真正碳交易机制）  ← 连接 Q1 和 Q2
步骤3：ε-约束法生成 Pareto 前沿                   ← 三目标权衡
步骤4：构建边际减排成本曲线（影子价格）            ← 策略决策依据
步骤5：三种代表性策略对比 + 运行建议               ← 综合策略输出
```

---

## 3. 步骤1：计算碳减排潜力上限

### 方法

二分搜索法寻找**无甩负荷前提下的最小可行碳排放**：

```
1. 上界 = 自然排放（无约束最优解的实际排放，42,237 kg）
2. 下界 = 自然排放的 50%
3. 二分收紧 cap，求解 S5（price=0）
4. 检查 unmet_load：若 < 1e-3 → cap 可行，取实际排放作为新上界；否则放宽下界
5. 收敛到无甩负荷的最小排放
```

区别于直接算"LP 可行性"——在极低 cap 下 LP 可以通过甩负荷变量找到"可行解"，但无实际意义。

### 结果

| 指标 | 值 |
|------|------|
| 自然排放（无约束） | 42,237 kg CO₂/周 |
| 最小可行排放（无甩负荷） | 34,357 kg CO₂/周 |
| **最大减排潜力** | **7,880 kg（18.7%）** |

---

## 4. 步骤2：cap × price 联合场景（真正碳交易）

### 为什么必须补充

第1小问（price=0）是命令控制，第2小问（cap=∞）是碳税。**两者都不是碳交易**。碳交易必须 cap 和 price 同时存在：

- cap（免费配额）→ 你能免费排多少
- price（交易价格）→ 超额买、结余卖的单价

### 方法

```
固定免费配额 cap = E_nat × 93%（39,281 kg）
扫描碳价 0/100/300/500 元/吨
求解 S5（price=碳价, cap=配额）
碳交易成本 = price × (实际排放 − cap)
提取碳约束的影子价格（边际减排成本）
```

### 结果

| 碳价 元/吨 | 排放 kg | vs 配额 | 交易行为 | 交易成本 元 | 影子价格 元/吨 |
|------:|------:|------:|------:|------:|------:|
| 0 | 39,281 | = 配额 | 不交易 | −0 | **917** |
| 100 | 39,281 | = 配额 | 不交易 | −0 | 817 |
| 300 | 39,281 | = 配额 | 不交易 | −0 | 617 |
| 500 | 39,281 | = 配额 | 不交易 | −0 | 417 |

### 核心发现

**四个碳价下排放完全不变**。原因是该配额水平下的**纯运营边际减排成本 = 917 元/吨**，所有测试碳价均低于此值：

```
碳价 < 917 元/吨 → 减排不如买配额 → 排放维持 = cap
碳价 > 917 元/吨 → 主动减排并出售结余配额获利
```

影子价格与碳价的线性关系验证：

```
shadow_price(综合) = 纯运营边际减排成本 − carbon_price

price=0:   0.917 = 0.917 − 0      ✓
price=100: 0.817 = 0.917 − 0.10   ✓
price=300: 0.617 = 0.917 − 0.30   ✓
price=500: 0.417 = 0.917 − 0.50   ✓
```

这个影子价格是**连接 Q1 和 Q2 的核心桥梁**——Q1 中硬约束的"影子成本"决定了 Q2 中碳价在什么水平上才会对调度产生实际影响。

## 5. 步骤3：Pareto 前沿分析

### 方法

ε-约束法：在 [最小可行排放, 自然排放] 区间均匀取 10 个 cap 值，求解 S5（price=0），过滤有甩负荷的无效点。

```
对每个 cap ∈ [34,357, 42,237]：
    求解 S5（price=0, cap=cap值）
    若 unmet_load < 1e-3 → 记录为有效 Pareto 点
    记录：碳排放、运行成本、寿命损耗、综合成本、影子价格
```

### Pareto 前沿数据

选取三种代表性策略：

| 策略 | 碳排放 kg | 运行成本 元 | 寿命损耗 元 | 综合成本 元 |
|------|------:|------:|------:|------:|
| 经济优先 | 42,237 | 44,485 | 973 | 45,458 |
| 均衡策略 | 38,735 | 46,974 | 969 | 47,943 |
| 碳优先 | 34,357 | 64,507 | 1,030 | 65,537 |

### 关键指标

| 指标 | 值 |
|------|------|
| 减排总额外成本（经济优先 → 碳优先） | 20,079 元 |
| **平均边际减排成本** | **2,548 元/吨** |
| 93% 配额水平下边际减排成本 | 917 元/吨 |
| 中国碳市场参考价 | ~100 元/吨 |
| EU ETS 参考价 | ~500 元/吨 |

### 分析

1. **边际成本递增**：浅层减排便宜，深度减排代价高昂。边际减排成本从接近 0 元/吨（近自然排放）攀升到远高于碳市场价格的数千元/吨。

2. **碳价全部低于边际减排成本**：无论中国碳市场（~100 元/吨）还是 EU ETS（~500 元/吨），均远低于园区平均边际减排成本（2,548 元/吨）乃至 93% 配额水平的边际成本（917 元/吨）。因此**园区在任何现实碳价下都应购买配额而非主动减排**。

3. **电池寿命基本不受影响**：寿命损耗从 973 元仅增至 1,030 元（+5.9%），说明减排主要通过减少购电实现，未以牺牲电池为代价。经济和电池两个目标之间基本不冲突。

## 6. 步骤4：边际减排成本曲线

用各 Pareto 点的碳约束影子价格，绘制边际减排成本（元/吨）随碳排放变化的曲线。

```
影子价格 = ∂(综合成本) / ∂(cap)
         = 每多允许排放 1 kg CO₂，综合成本下降多少元
```

该曲线反映园区在不同排放水平下的减排代价，并标注中国碳市场和 EU ETS 参考价线，直观展示碳价与边际减排成本的差距。

## 7. 三个维度的关系总结

| 关系 | 结论 |
|------|------|
| 经济性 vs 碳排放 | **强权衡关系**。减排 18.7% 需多花 20,079 元（+44%） |
| 电池寿命 vs 碳排放 | **基本不冲突**。寿命损耗仅增 5.9%，减排不依赖电池过度循环 |
| 经济性 vs 电池寿命 | 从问题二已知存在权衡，但量级远小于碳减排的代价 |
| 碳价信号 vs 碳配额 | **配额约束力 >> 价格信号**。碳价需超过边际减排成本阈值（~917 元/吨）才生效 |

三者通过 S5 综合成本目标函数 `min 运行成本 + 寿命损耗 + 碳交易成本` 自动统一优化。

## 8. 综合运行策略建议

### 8.1 核心决策准则

```
比较 "碳市场碳价" vs "园区边际减排成本"

碳价 < 边际减排成本 → 购买配额，不做过度减排
碳价 > 边际减排成本 → 主动减排，出售结余配额获利
```

### 8.2 策略建议

1. **常态操作（碳价 < 917 元/吨时）**：
   - 以 S5 优化日常调度，将碳配额作为硬约束
   - 通过减少高碳强度时段购电、利用储能时移、建筑负荷调节实现减排
   - **优先购买碳配额**满足合规，不做超越配额要求的额外减排
   - EV V2B 以电池寿命保护为主，不用于碳减排

2. **碳价上涨时（碳价 > 917 元/吨时）**：
   - 启动主动减排：进一步减少购电 + 增加储能调节 + 建筑柔性负荷
   - **出售结余配额获利**
   - EV V2B 仍最后考虑（电池损耗代价最高）

3. **减排方式优先序**：
   ```
   减少外网购电 → 固定储能调节 → 建筑负荷时移/削减 → EV V2B（最后手段）
   ```

4. **动态碳资产管理**：
   - 定期更新边际减排成本曲线
   - 跟踪碳市场价格，在低价时储备配额、高价时出售结余
   - 若允许配额跨期存储（banking），在碳价低谷期购入配额储备

---

## 9. 核心结论

```
1. 碳配额硬约束是唯一有效的减排工具。
   碳价 500 元/吨仅减排 1.4%，而 93% 配额可减排 7%。

2. 园区边际减排成本（917–2,548 元/吨）远高于现实碳价。
   在任何现实碳市场中，园区都应购买配额而非主动减排。

3. 电池寿命与碳排放之间基本不冲突。
   减排不依赖电池过度循环，经济性和电池寿命的权衡在碳减排场景下不构成主要矛盾。

4. 碳约束的影子价格（917 元/吨）是连接碳配额和碳价的核心桥梁。
   它决定了碳价在什么水平上才会对调度产生实际影响。

5. 碳交易机制（cap + price 同时存在）的正确运作方式：
   常态下买配额满足合规，仅在碳价突破边际减排成本时才主动减排并出售配额。
```
