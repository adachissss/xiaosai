# B 题第二问建模计划

## 1. 第二问任务理解

第二问是在第一问已经建立“源—荷—储—车”协同运行模型的基础上，进一步把固定储能和电动车电池的寿命损耗纳入调度决策。

题目要求完成 5 个方面：

1. 提出衡量固定储能与电动车电池寿命损耗的指标或成本折算方式；
2. 在运行方案中同时考虑运行成本与电池寿命损耗，重新设计调度安排；
3. 比较引入电池寿命损耗前后，系统调度行为的主要变化；
4. 讨论固定储能设备与电动车在削峰、储能和供能中各自应承担的角色与分工；
5. 分析短期经济收益与长期电池健康损耗之间如何权衡。

因此，第二问不是重新建立一个完全不同的系统，而是在第一问 S3 完整协同优化模型上加入“电池寿命损耗成本”，然后比较加入前后的差异。

## 4. 第二问分步推进方式

第二问有多个小要求，不建议一次性全部写完。建议按以下顺序逐步完成。

### Step 1：建立电池寿命损耗指标

对应题目第 1 小点。

目标：先定义固定储能和 EV 的寿命损耗如何计算。

建议采用“单位能量吞吐损耗成本”模型：

```text
电池寿命损耗成本 = 单位吞吐损耗成本 × 充放电吞吐量
```

固定储能吞吐量：

```text
E_bat_throughput = Σ_t (P_bat_ch[t] + P_bat_dis[t]) Δt
```

固定储能损耗成本：

```text
C_bat_deg = c_bat_deg × E_bat_throughput
```

EV 吞吐量：

```text
E_ev_throughput = Σ_i Σ_t (P_ev_ch[i,t] + P_ev_dis[i,t]) Δt
```

EV 损耗成本：

```text
C_ev_deg = Σ_i c_ev_deg[i] × Σ_t (P_ev_ch[i,t] + P_ev_dis[i,t]) Δt
```

其中：

- `c_bat_deg`：固定储能单位吞吐损耗成本；
- `c_ev_deg[i]`：第 `i` 辆 EV 的单位吞吐损耗成本。

EV 的 `c_ev_deg[i]` 使用 `ev_sessions.csv` 中的 `degradation_cost_cny_per_kwh_throughput` 字段。

固定储能的 `c_bat_deg` 使用 `asset_parameters.csv` 中的给定字段：

```text
stationary_battery_degradation_cost_cny_per_kwh_throughput = 0.055 元/kWh
```

因此第二问第 1 小问不需要再人为假设固定储能寿命成本参数。

### Step 2：建立考虑寿命损耗的 S4 优化模型

对应题目第 2 小点。

在第一问 S3 的目标函数中加入：

```text
C_bat_deg + C_ev_deg
```

新的目标函数为：

```text
min 运行成本 + 固定储能寿命损耗成本 + EV 寿命损耗成本
```

也就是：

```text
min C_grid
  + C_shed
  + C_shift
  + C_curtail
  + C_peak
  + C_bat_deg
  + C_ev_deg
```

其中：

```text
C_bat_deg = c_bat_deg Σ_t (P_bat_ch[t] + P_bat_dis[t]) Δt
C_ev_deg = Σ_i c_ev_deg[i] Σ_t (P_ev_ch[i,t] + P_ev_dis[i,t]) Δt
```

注意：第一问中为减少线性规划退化，已经加入极小吞吐惩罚 `0.001 × Δt`。第二问中这个极小惩罚应被正式的寿命损耗成本替代或覆盖，不应重复解释为寿命成本。

### Step 3：比较引入寿命损耗前后调度变化

对应题目第 3 小点。

主要比较 S3 和 S4：

| 指标 | S3 未考虑寿命 | S4 考虑寿命 | 变化 |
|---|---:|---:|---:|
| 总运行成本 |  |  |  |
| 电网购电费用 |  |  |  |
| 固定储能吞吐量 |  |  |  |
| EV 充电吞吐量 |  |  |  |
| EV 放电吞吐量 |  |  |  |
| 固定储能寿命损耗成本 |  |  |  |
| EV 寿命损耗成本 |  |  |  |
| 峰值购电功率 |  |  |  |
| 光伏消纳率 |  |  |  |
| 建筑负荷转移量 |  |  |  |
| 建筑负荷削减量 |  |  |  |

预期变化：

- 固定储能和 EV 的充放电频率下降；
- EV V2B 放电会更加谨慎；
- 系统可能更多依赖外网购电或建筑柔性负荷；
- 短期电费可能上升；
- 电池寿命损耗成本下降；
- 总综合成本可能更合理。

### Step 4：讨论固定储能与 EV 的角色分工

对应题目第 4 小点。

建议论文中给出以下观点：

固定储能更适合承担：

- 光伏消纳；
- 日内削峰填谷；
- 园区内部稳定调节；
- 频率较高、计划性较强的充放电任务。

EV 更适合承担：

- 在不影响离站需求的前提下参与辅助调节；
- 高价时段或购电受限时段的短时 V2B；
- 作为补充灵活性资源，而不是主要储能设备；
- 对 EV 电池寿命损耗应给予更高补偿或更强约束。

### Step 5：分析短期经济收益与长期电池健康权衡

对应题目第 5 小点。

核心分析逻辑：

- 如果不考虑寿命损耗，优化模型会倾向于频繁使用电池，只要能降低电费即可；
- 如果考虑寿命损耗，只有当节省的购电成本大于电池损耗成本时，充放电才是合理的；
- 对固定储能，园区可以在收益和寿命之间做统一权衡；
- 对 EV，电池属于车主资产，V2B 应谨慎使用，并需要补偿机制；
- 长期运行策略应避免为了短期电费收益牺牲电池健康。

## 5. 寿命损耗模型选择

第二问可以有复杂模型，也可以有简化模型。考虑数学建模比赛和当前数据情况，建议采用线性吞吐量损耗模型。

### 5.1 推荐模型：线性吞吐量损耗

优点：

- 仍然保持线性规划结构；
- 易于解释；
- 易于和第一问代码衔接；
- 适合题目给定的数据粒度。

形式：

```text
C_deg = c_deg × Σ 充放电吞吐量
```

这里的“吞吐量”可以理解为电池经历的总能量流动。

### 5.2 不建议一开始采用复杂非线性寿命模型

例如：

- SOC 深度相关的非线性循环寿命模型；
- 雨流计数法；
- 温度相关退化模型；
- C-rate 相关退化模型。

原因：

- 题目数据没有电池温度、电池健康状态等详细参数；
- 非线性模型会让问题变成非线性规划或混合整数模型；
- 实现复杂度高，论文解释也更难；
- 不利于和第一问结果直接对比。

可以在论文讨论中说明：本模型采用线性等效吞吐损耗作为近似，复杂老化机理可作为后续扩展。

## 6. 固定储能寿命成本参数建议

附件已经给出了固定储能和 EV 的单位吞吐寿命成本，因此本题优先使用数据表参数，不再额外估算。

固定储能寿命成本字段位于：

```text
asset_parameters.csv
```

字段为：

```text
stationary_battery_degradation_cost_cny_per_kwh_throughput = 0.055 元/kWh
```

EV 寿命成本字段位于：

```text
ev_sessions.csv
```

字段为：

```text
degradation_cost_cny_per_kwh_throughput
```

这样固定储能和 EV 都可以用统一公式计算：

```text
C_deg = c_deg × E_throughput
```

后续如需分析参数不确定性，可以围绕表中给定值做敏感性分析，例如固定储能取 `0.8倍、1.0倍、1.2倍`。

## 9. 第二问论文写作框架

### 9.1 寿命损耗指标建立

先解释为什么需要考虑寿命损耗：

- 电池频繁充放电会减少循环寿命；
- 第一问只追求短期运行成本，可能过度使用储能和 EV；
- 第二问要把短期收益和长期健康统一进目标函数。

然后给出线性吞吐量损耗模型。

### 9.2 考虑寿命损耗的优化模型

说明在第一问 S3 模型的基础上，目标函数增加：

```text
C_bat_deg + C_ev_deg
```

约束保持基本不变。

### 9.3 结果对比

重点比较：

- S3：不考虑寿命损耗；
- S4：考虑寿命损耗。

分析调度变化：

- 电池吞吐量；
- EV V2B；
- 外网购电；
- 峰值购电；
- 建筑柔性负荷；
- 综合成本。

### 9.4 角色分工

说明固定储能和 EV 不应承担完全相同的功能：

- 固定储能是园区资产，适合承担主要调节；
- EV 是用户资产，适合作为补充灵活性资源；
- EV 参与 V2B 时需要满足出行需求并补偿电池损耗。

### 9.5 权衡分析

说明：

```text
当电价差收益 > 电池寿命损耗成本时，充放电是合理的；
当电价差收益 < 电池寿命损耗成本时，应减少电池使用。
```

这就是短期经济收益和长期电池健康之间的权衡原则。

## 11. 当前推荐结论

第二问最终可以形成这样的结论方向：

1. 引入寿命损耗后，模型会减少不必要的固定储能和 EV 充放电；
2. EV V2B 的使用会更加集中在高价值时段，例如高电价或购电受限时段；
3. 固定储能更适合承担主要削峰填谷任务；
4. EV 更适合作为补充灵活性资源，不宜频繁深度参与园区供电；
5. 虽然考虑寿命后短期购电成本可能上升，但综合长期成本更加合理；
6. 园区应通过寿命损耗补偿机制，引导 EV 在不损害车主利益的前提下参与协同运行。
