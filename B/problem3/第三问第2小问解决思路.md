# 第三问第 2 小问解决思路：不同碳交易价格对系统调度的影响

## 1. 小问目标

第三问第 2 小问要求：

```text
分析碳交易价格水平如何影响园区从外部购电、储能充放电及电动车 V2B 决策。
```

与第 1 小问不同，本小问不设碳排放硬上限（cap = None），仅通过碳交易价格信号（加入目标函数）影响系统决策。这模拟的是纯粹的碳市场机制——排放越多，碳交易成本越高，但排放量本身不受硬性限制。

---

## 2. 模型基础

S5 模型的目标函数中，碳交易成本被直接加入：

```text
min 运行成本 + 固定储能寿命损耗 + EV 寿命损耗 + 碳交易成本
```

其中碳交易成本为：

```text
C_carbon = carbon_price × Σ_t(grid_buy[t] × carbon_intensity[t] × Δt)
     = carbon_price × Total_Carbon
```

在 LP 中实现为 `grid_buy[t]` 变量的线性系数叠加：

```python
c[grid_buy[t]] += carbon_price × carbon_intensity[t] × Δt
```

这样，购电不仅支付电费（`c[grid_buy[t]] = 购电价 × Δt`），还要支付对应的碳成本。两者合并后的"有效购电成本"为：

```text
effective_grid_cost[t] = (购电价 + carbon_price × carbon_intensity[t]) × Δt
```

碳价越高，高碳强度时段的购电成本相对上升越多，LP 倾向于在高碳时段减少购电。

## 3. 碳价情景设计

设定四种碳价级别，不设碳排放硬上限：

| 场景 | carbon_price（元/kg） | carbon_price（元/吨） | 含义 |
|---:|---:|---:|---|
| 无碳成本 | 0.00 | 0 | 基线——不考虑碳成本 |
| 低碳价 | 0.10 | 100 | 参照当前国内碳市场水平（约 60-120 元/吨） |
| 中碳价 | 0.30 | 300 | 模拟碳市场较高水平 |
| 高碳价 | 0.50 | 500 | 模拟碳市场极端水平或附加惩罚 |

## 4. 关键代码说明

### 4.1 碳交易成本加入目标函数

在 `carbon_aware_strategy.py:117-119`：

```python
carbon_intensity = ts["grid_carbon_kg_per_kwh"].to_numpy()
c[grid_buy] += carbon_price_cny_per_kg * carbon_intensity * DT
```

含义：每个时段 `grid_buy[t]` 的目标系数增加 `碳价 × 碳强度 × 步长`。这等价于对购电征收与碳强度成正比的"碳税"。在 LP 看来，不同的 carbon_price 会改变购电在不同时段的相对成本，从而影响购电时段的分布决策。

### 4.2 场景遍历

在 `solve_problem3.py:282-301`：

```python
for label, price in price_scenarios:
    sched, ev_r, info = build_and_solve_carbon_aware(
        data, carbon_price_cny_per_kg=price, carbon_cap_kg=None, scheme_name=f"S5_price_{label}"
    )
    m = extract_metrics(sched, ev_r, data, data_dir, price, None)
    m["scenario_label"] = label
    metrics_list.append(m)
```

含义：对每个碳价调用 S5 求解器，不设碳排放上限。提取完整的指标（碳排放、运行成本、碳交易成本、储能吞吐量、EV 放电等）进行对比。

### 4.3 碳交易成本计算

在 `solve_problem3.py:87`：

```python
carbon_trading_cost = total_carbon_kg * carbon_price
```

含义：碳交易成本 = 实际碳排放 × 碳价。在 S5 中，这部分成本已被纳入目标函数，因此 LP 求解时会自动权衡减排和碳税成本的关系。

## 6. 结果分析

### 6.1 碳价对排放的抑制效果有限

即使碳价升至 500 元/吨（远超当前国内碳市场水平），碳排放仅从 42237 kg 降到 41651 kg，降幅仅 **1.4%**。

这是本小问最重要的发现，原因有三：

1. **园区负荷需求刚性较强**：办公建筑、湿实验室、教学中心的用电需求无法大幅削减，总购电量难以显著下降。
2. **碳强度波动范围有限**：grid_carbon 在 0.55-0.74 kg/kWh 之间，最高时段仅比最低时段高 35%。即使完全避开高碳时段，节省的碳排放也有限。
3. **储能和光伏的调节能力已接近饱和**：在 S4 基线中，固定储能吞吐量已达 12952 kWh，光伏消纳率 100%，没有更多"无碳资源"可以调动来替代购电。

### 6.2 碳价主要影响成本结构而非调度行为

碳交易成本随碳价线性增长：

| 碳价 | 碳交易成本 元 | 占总综合成本比例 |
|---:|---:|---:|
| 0 | 0 | 0% |
| 100 | 4213 | 8.5% |
| 300 | 12624 | 21.7% |
| 500 | 20825 | 31.3% |

碳交易成本占总成本的比例从 0 上升到 31.3%，这虽然是财务上的重大变化，但对**调度行为的边际影响**很小。因为在 LP 的结构下，目标函数中各成本项的梯度决定了调度决策；碳价的梯度（来自碳强度）相对于购电价的梯度较小，不足以根本改变购电的时段分配。

### 6.3 EV 放电完全不变

在所有碳价下，EV 放电量均保持 **433.9 kWh**，没有变化。

```text
碳价 0   → EV 放电 433.9 kWh
碳价 100 → EV 放电 433.9 kWh
碳价 300 → EV 放电 433.9 kWh
碳价 500 → EV 放电 433.9 kWh
```

原因分析：
1. EV 放电主要通过 V2B 在高峰时段替代外网购电
2. EV 放电本身就面临较高的单位寿命损耗成本，只有在替代购电的收益大于损耗成本时才会被调度
3. 碳价增加后，购电成本确实上升了，但 EV 主要在高价/高峰时段放电，这些时段恰好是常规购电价已经很高的时段，碳强度不一定最高
4. EV 放电带来的碳排放减少主要通过"减少购电"间接实现，但减少的购电量很小（433.9 kWh → 433.9 kWh 无变化），说明 LP 没有因为碳价而额外调用 V2B

### 6.4 购电量的小幅调整

| 碳价 | 购电总量 kWh | 购电峰值 kW |
|---:|---:|---:|
| 0 | 63990 | 608 |
| 100 | 63986 | 606 |
| 300 | 63973 | 598 |
| 500 | 63527 | 580 |

随碳价上升，购电总量仅小幅下降（-0.7%），购电峰值从 608 kW 降至 580 kW（-4.6%）。说明碳价机制对峰值购电的抑制效果略好于对总量的抑制，因为高碳强度时段往往与购电高峰时段重合（火电机组在高峰期出力占比高）。

### 6.5 固定储能在高碳价下吞吐量下降

碳价 500 元/吨时，固定储能吞吐量从 12952 kWh 降至 12902 kWh（-0.4%）。这与第 1 小问的发现一致：碳价使得从高碳强度时段充电的成本更高，储能套利空间缩小。

### 6.6 负荷转移的调整

| 碳价 | 负荷转移下调 kWh | 负荷削减 kWh |
|---:|---:|---:|
| 0 | 5650 | 693 |
| 100 | 5601 | 693 |
| 300 | 5403 | 693 |
| 500 | 4771 | 1093 |

随着碳价上升，系统主动减少了负荷转移下调（即较少将负荷转移到高碳时段），同时在高碳价（500）时增加了负荷削减（从 693 到 1093 kWh）。这表明碳价机制促使系统在高碳时段选择削负荷而非转移负荷。

## 8. 第 1 小问与第 2 小问的结果对比

两种机制的比较：

| 维度 | 碳排放上限（第 1 小问） | 碳交易价格（第 2 小问） |
|---:|---|
| 机制类型 | 硬约束（行政手段） | 软信号（市场手段） |
| 减排效果 | 显著——cap 收紧到 86% 可减排 14% | 有限——碳价 500 元/吨仅减排 1.4% |
| 成本影响 | 运行成本上升 13.4% | 综合成本上升 46%，但主要是碳税成本转账 |
| 调度行为变化 | 明显——购电、负荷削减全部调整 | 较小——仅在高碳价时有微调 |
| 机制优劣 | 调控精准但缺乏弹性 | 经济效率高但当前参数下调控力度不足 |

## 9. 本小问结论

第三问第 2 小问已完成：

```text
在 S5 模型中通过不同的碳交易价格（0/100/300/500 元/吨），分析了价格信号
对外网购电、储能充放电和 EV V2B 决策的影响。
```

核心发现：
1. 碳价对碳排放的边际抑制效果很弱——500 元/吨仅减排 1.4%
2. 原因是园区可调资源（储能、光伏）已接近物理上限，碳价信号难以创造额外的无碳替代
3. EV V2B 放电量在所有碳价下完全不变，说明碳价不改变 V2B 的调度决策
4. 碳价的主要影响体现在成本结构上——碳交易成本在 500 元/吨时占总成本的 31%
5. 碳价机制在园区层面单独使用的调控效果有限，需要与碳排放上限配合使用
