# 第二问第 4 小问解决思路：固定储能与电动车的角色分工

## 1. 小问目标

第二问第 4 小问要求：

```text
讨论固定储能设备与电动车在削峰、储能和供能中各自应承担的角色与分工。
```

这一小问同样以讨论为主，不需要再建立新的优化模型。它应当基于前面 S3/S4 的计算结果，结合固定储能和 EV 的物理属性、资产属性、调度约束和寿命损耗差异，说明两类电池资源在园区能源系统中不应承担完全相同的功能。

核心问题是：

```text
固定储能和 EV 都可以充放电，但它们在系统中应该如何分工？
```

本文建议形成如下结论：

```text
固定储能应作为园区内稳定、可控、计划性的主调节资源；
EV 应作为满足出行需求前提下的补充灵活性资源，适合短时、低频、高价值地参与削峰和应急供能。
```

---

## 2. 分析思路

### 2.1 为什么需要讨论角色分工

从数学模型看，固定储能和 EV 都可以被抽象为可充放电电池，形式上都具有：

```text
充电变量 P_ch
放电变量 P_dis
电量状态 SOC
功率约束
容量约束
```

但是从实际系统运行看，两者存在明显差异：

| 对比维度 | 固定储能 | 电动车 EV |
|---|---|---|
| 资产属性 | 园区资产 | 用户/车主资产 |
| 是否专门用于能源调节 | 是 | 否，首要功能是出行 |
| 可调度时间 | 全时段可调度 | 仅停车接入期间可调度 |
| 调度可控性 | 高 | 受用户到离站时间影响 |
| 离站需求 | 无 | 必须满足离站电量需求 |
| 寿命损耗承担者 | 园区或运营方 | 车主或 EV 用户 |
| 适合承担任务 | 高频、计划性调节 | 低频、补充性调节 |

因此，如果模型不区分两者角色，可能会出现不合理现象：

```text
为了短期电费收益频繁调用 EV 电池，损害车主利益和电池健康。
```

所以，第 4 小问应重点讨论：

1. 固定储能适合承担哪些任务；
2. EV 适合承担哪些任务；
3. 在削峰、储能和供能中如何分工；
4. 为什么这种分工更符合长期运行逻辑。

---

### 2.2 结合前面结果进行讨论

第 3 小问中已经发现：

| 指标 | S3 未考虑寿命 | S4 考虑寿命 | 变化 |
|---|---:|---:|---:|
| 固定储能吞吐量/kWh | 12952.421 | 12952.421 | 0.000 |
| EV 总吞吐量/kWh | 3349.736 | 3015.055 | -334.682 |
| EV 放电量/kWh | 592.676 | 433.911 | -158.765 |
| EV 寿命损耗成本/元 | 288.246 | 260.826 | -27.421 |

这说明：

1. 考虑寿命损耗后，固定储能仍保持原有调节强度；
2. EV 的充放电吞吐量明显下降；
3. EV 的 V2B 放电被明显削减；
4. 模型更倾向于让固定储能承担主要调节任务，让 EV 承担补充调节任务。

因此，第 4 小问的讨论应当以这个结果为支撑，而不是纯理论判断。

---

## 3. 固定储能应承担的角色

### 3.1 固定储能适合作为园区主调节资源

固定储能是安装在园区内部的专用储能设备，它的主要目的就是参与能源系统调节。因此，它适合作为园区协同运行中的主调节资源。

固定储能的优势包括：

1. **可控性强**  
   固定储能不会像 EV 一样受车辆到站、离站时间影响，调度中心可以在整个调度周期内连续控制其充放电。

2. **运行目标明确**  
   固定储能的功能就是服务园区能源系统，可以直接围绕削峰填谷、光伏消纳和购电成本优化进行调度。

3. **责任主体清晰**  
   固定储能属于园区或运营方资产，寿命损耗成本可以直接纳入园区运营成本进行统一权衡。

4. **适合高频调节**  
   相比 EV，固定储能更适合承担频率较高、计划性较强的日内充放电任务。

因此，固定储能在系统中的定位应为：

```text
园区内部稳定、可控、计划性的主储能资源。
```

---

### 3.2 固定储能在削峰中的作用

削峰是指在负荷高峰或电价高峰时段，通过储能放电减少外网购电功率。

固定储能适合承担主要削峰任务，原因是：

1. 它不受车辆停留时间限制；
2. 可提前在低价或光伏富余时段充电；
3. 可在高峰时段按计划放电；
4. 调度结果稳定，可预测性强。

在当前结果中，S3 和 S4 的峰值购电功率均为：

```text
607.550 kW
```

固定储能吞吐量保持：

```text
12952.421 kWh
```

这说明即使加入寿命损耗成本，模型仍然保持固定储能的调节强度，说明其削峰收益足以覆盖寿命损耗。

因此，在削峰任务中，固定储能应承担：

```text
基础削峰、日内峰谷转移和常规高峰供能。
```

---

### 3.3 固定储能在储能中的作用

储能功能主要体现在低价时段或光伏富余时段充电，高价或负荷高峰时段放电。

固定储能适合承担主要储能任务，因为它具有固定容量、固定功率和全天候可用性。

尤其在新能源园区中，固定储能可以用于：

1. 吸收中午光伏出力；
2. 减少弃光；
3. 将光伏电量转移到傍晚或夜间使用；
4. 平滑园区净负荷曲线。

当前 S3 和 S4 光伏消纳率均为：

```text
1.000
```

说明园区能够完全消纳光伏。固定储能在其中起到了重要支撑作用。

因此，在储能功能中，固定储能应承担：

```text
光伏消纳、日内能量搬移和园区内部能量缓冲。
```

---

### 3.4 固定储能在供能中的作用

供能是指在负荷高、电价高或外网受限时，向园区负荷释放电能。

固定储能由于可控性强，适合作为园区供能的稳定补充来源。

它尤其适合：

1. 高价时段替代部分外网购电；
2. 峰值约束紧张时提供功率支撑；
3. 光伏不足但负荷较高时补充供能；
4. 在不影响用户出行的情况下提供内部供电保障。

因此，在供能任务中，固定储能应承担：

```text
常规供能补充和高峰时段稳定支撑。
```

---

## 4. EV 应承担的角色

### 4.1 EV 首要属性是交通工具，不是专用储能

EV 虽然具有电池和双向充放电能力，但它的首要功能是满足用户出行需求。

EV 调度必须优先满足：

```text
车辆在离站时达到用户要求电量。
```

因此，EV 不能像固定储能一样被无限制地频繁调用。

EV 参与园区调度时，需要同时满足：

1. 车辆必须在站；
2. 车辆允许 V2B；
3. 不影响离站电量需求；
4. 放电收益应覆盖电池寿命损耗；
5. 需要考虑车主补偿机制。

因此，EV 在系统中的定位应为：

```text
满足出行需求前提下的补充灵活性资源。
```

---

### 4.2 EV 在削峰中的作用

EV 可以参与削峰，但不应作为主要削峰资源。

EV 适合参与的削峰场景包括：

1. 高电价时段；
2. 负荷尖峰持续时间较短；
3. 固定储能功率不足；
4. EV 停车时间充足；
5. EV 离站电量仍能满足要求；
6. V2B 收益足以覆盖寿命损耗和车主补偿。

也就是说，EV 更适合承担：

```text
短时尖峰削减和高价值时段补充削峰。
```

第 3 小问结果显示，引入寿命损耗后，EV 放电量从：

```text
592.676 kWh
```

下降到：

```text
433.911 kWh
```

说明模型减少了低收益的 EV 放电，只保留更有价值的 V2B 调用。

这正好说明 EV 在削峰中的合理定位：

```text
不是承担常规削峰主力，而是在高价值时段提供补充削峰能力。
```

---

### 4.3 EV 在储能中的作用

EV 理论上可以作为分布式储能资源，但其储能能力具有不确定性。

原因包括：

1. EV 到站时间不固定；
2. EV 离站时间不固定；
3. 每辆车初始电量不同；
4. 每辆车离站需求不同；
5. 并非所有 EV 都允许 V2B；
6. EV 电池寿命损耗由车主承担。

因此，EV 不适合作为园区主要储能资源。

EV 更适合承担：

```text
分散式、机会型、短时储能。
```

例如：

- 在中午光伏富余且车辆停留时间较长时，适度提高 EV 充电；
- 在傍晚高价时段，允许部分满足条件的 EV 短时放电；
- 对寿命成本较低、离站时间较晚、V2B 允许的车辆优先调用。

因此，EV 储能应遵循：

```text
优先满足出行需求，其次参与系统调节。
```

---

### 4.4 EV 在供能中的作用

EV 可以向园区供能，但应谨慎使用。

适合 EV 供能的场景包括：

1. 外网购电价格较高；
2. 园区短时负荷尖峰较大；
3. 固定储能出力不足；
4. EV 停留时间较长；
5. EV 离站需求仍可满足；
6. 车主能获得合理补偿。

EV 不适合承担长时间、连续性、基础性的供能任务。

原因是：

```text
EV 是用户资产，频繁放电会增加车主电池寿命损耗，并可能影响出行体验。
```

因此，在供能任务中，EV 应承担：

```text
短时应急供能、高价时段补充供能和固定储能不足时的辅助供能。
```

---

## 5. 固定储能与 EV 的分工总结

可以将二者分工概括如下。

| 功能 | 固定储能 | EV |
|---|---|---|
| 削峰 | 承担基础削峰和常规峰谷调节 | 承担短时尖峰和高价值时段补充削峰 |
| 储能 | 承担光伏消纳、日内能量搬移和能量缓冲 | 承担机会型、分散式、短时储能 |
| 供能 | 承担高峰时段稳定供能补充 | 承担短时辅助供能和应急供能 |
| 调度频率 | 可较高频调度 | 应低频、谨慎调度 |
| 调度优先级 | 优先作为主调节资源 | 在满足出行需求后作为补充资源 |
| 寿命成本处理 | 园区统一计入运营成本 | 应计入车主补偿或 V2B 激励 |

因此，推荐分工为：

```text
固定储能承担“主储能、主削峰、稳定供能”；
EV 承担“补充储能、补充削峰、短时供能”。
```

## 6. 与 S4 结果的对应关系

S4 的结果支持上述分工。

S3 和 S4 固定储能吞吐量均为：

```text

```

说明即使考虑寿命损耗，固定储能仍然被充分调用。

这表明在当前参数下，固定储能的调节收益较高，适合作为园区主要调节资源。

### 6.2 EV 调用更加谨慎

EV 总吞吐量从：

```text
3349.736 kWh
```

下降到：

```text
3015.055 kWh
```

EV 放电量从：

```text
592.676 kWh
```

下降到：

```text
433.911 kWh
```

说明引入寿命损耗后，模型主动减少了 EV 的充放电循环和 V2B 调用。

这表明 EV 更适合作为补充灵活性资源，而不是主要储能设备。

### 6.3 系统运行质量没有明显下降

S4 中：

- 光伏消纳率仍为 `1.000`；
- 峰值购电功率仍为 `607.550 kW`；
- 建筑负荷转移量和削减量基本不变；
- EV 离站缺口没有进一步恶化。

说明减少 EV 的低价值调用后，系统仍能保持良好运行。

这进一步说明：

```text
EV 并不需要承担主要调节任务，固定储能和建筑柔性负荷已经可以支撑大部分系统调节需求。
```

## 7. 论文表达建议

论文中可以这样表述：

```text
固定储能与电动车虽然都具有充放电能力，但二者在资产属性、可调度性和运行目标上存在明显差异。固定储能是园区专用能源设备，具有较强的可控性和连续调度能力，适合承担光伏消纳、日内能量搬移、基础削峰和高峰供能等主要调节任务。相比之下，电动车的首要功能是满足用户出行需求，其可调度时间受车辆到离站约束影响，且电池寿命损耗直接关系到车主利益，因此不宜作为园区主要储能资源。电动车更适合作为补充灵活性资源，在高电价、短时尖峰或固定储能出力不足等高价值场景下参与 V2B。本文 S4 结果也表明，引入寿命损耗后，固定储能吞吐量基本不变，而 EV 总吞吐量和放电量明显下降，说明模型倾向于保留固定储能的主调节作用，同时减少 EV 的低收益充放电行为。因此，合理分工应为：固定储能承担主储能、主削峰和稳定供能，EV 在满足出行需求和补偿寿命损耗的前提下承担补充削峰、短时供能和辅助调节。
```

---

## 8. 本小问结论

第二问第 4 小问的核心结论为：

```text
固定储能应作为园区能源系统的主要调节资源，承担常规削峰、光伏消纳、日内储能和稳定供能；
EV 应作为补充灵活性资源，在满足出行需求和寿命补偿的前提下，参与短时削峰、高价值 V2B 和辅助供能。
```

这种分工既能发挥固定储能的稳定可控优势，又能避免过度消耗 EV 电池寿命，更符合新能源园区长期协同运行的要求。
