# 第二问第 3 小问解决思路：引入电池寿命损耗前后调度行为变化分析

## 1. 小问目标

第二问第 3 小问要求：

```text
比较引入电池寿命损耗前后，系统调度行为的主要变化。
```

这一小问以结果讨论为主，不需要重新建立新模型，而是基于第 2 小问已经得到的两个方案进行对比分析：

```text
S3_full_coordination：未考虑电池寿命损耗的完整协同方案
S4_degradation_aware：考虑电池寿命损耗的完整协同方案
```

S3 和 S4 的约束体系基本一致，区别在于目标函数不同：

```text
S3：min 运行成本
S4：min 运行成本 + 固定储能寿命损耗成本 + EV 寿命损耗成本
```

因此，本小问的核心分析逻辑是：

```text
在相同系统约束下，观察目标函数加入寿命损耗成本后，模型会如何改变固定储能、EV、外网购电、建筑柔性负荷和光伏消纳等调度行为。
```

---

## 2. 分析思路

### 2.1 为什么比较 S3 和 S4

第一问已经给出了多个方案：

- S0：朴素方案；
- S1：规则储能方案；
- S2：部分协同方案；
- S3：完整协同方案。

第二问第 2 小问是在 S3 的基础上加入寿命损耗成本形成 S4。因此，第 3 小问最直接、最公平的对比对象是：

```text
S3 vs S4
```

原因是：

1. 两者都允许固定储能优化调度；
2. 两者都允许 EV 智能充电和 V2B；
3. 两者都允许建筑柔性负荷参与调节；
4. 两者都使用相同的物理约束；
5. 唯一区别是 S4 在目标函数中加入了电池寿命损耗成本。

因此，如果 S4 相比 S3 出现调度变化，就可以认为这些变化主要来自寿命损耗成本的引入。

---

### 2.2 比较哪些指标

本小问不是只比较总成本，而是要说明“调度行为”发生了什么变化。因此需要从多个角度比较。

主要比较指标包括：

| 类型 | 指标 | 分析目的 |
|---|---|---|
| 成本指标 | 运行成本 | 判断短期经济性变化 |
| 成本指标 | 固定储能寿命损耗成本 | 判断固定储能使用强度变化 |
| 成本指标 | EV 寿命损耗成本 | 判断 EV 电池使用强度变化 |
| 成本指标 | 综合成本 | 判断加入寿命损耗后总体是否更优 |
| 固定储能行为 | 固定储能充放电吞吐量 | 判断储能是否减少循环使用 |
| EV 行为 | EV 充电量、放电量、总吞吐量 | 判断 EV 是否减少 V2B 和无效循环 |
| 电网行为 | 购电量、峰值购电功率 | 判断减少电池使用后是否增加对外网依赖 |
| 光伏行为 | 光伏消纳率 | 判断寿命成本是否牺牲光伏利用 |
| 建筑负荷行为 | 转移量、削减量 | 判断系统是否更多依赖负荷响应 |
| 约束指标 | EV 缺口、未满足负荷 | 判断新方案是否影响服务质量 |

其中最关键的指标是：

```text
EV 吞吐量、EV 放电量、EV 寿命损耗成本、综合成本。
```

因为当前结果中，寿命损耗引起的主要变化集中在 EV 侧。

---

### 2.3 分析逻辑

具体分析时按照以下顺序展开：

1. 先比较综合成本，判断 S4 是否在总体目标上优于 S3；
2. 再拆分运行成本和寿命损耗成本，解释为什么综合成本会变化；
3. 然后分析固定储能吞吐量，判断固定储能调度是否被削弱；
4. 重点分析 EV 充电、放电和吞吐量，判断 V2B 行为是否减少；
5. 再分析电网购电和峰值购电，判断减少 EV 放电是否导致外网依赖增加；
6. 接着分析光伏消纳和建筑柔性负荷，判断系统是否牺牲其他运行指标；
7. 最后总结调度行为变化背后的机制。

这种分析顺序可以避免只罗列数据，而是形成完整逻辑：

```text
目标函数变化 → 成本结构变化 → 电池资源调用变化 → 系统运行行为变化 → 调度机制解释
```

## 4. 调度行为变化分析

### 4.1 综合成本变化：S4 总体更优

从结果看，S4 的运行成本比 S3 增加：

```text
10.564 元
```

但是 S4 的寿命损耗成本减少：

```text
27.421 元
```

因此 S4 的综合成本反而降低：

```text
16.857 元
```

这说明，引入寿命损耗成本后，模型并不是继续追求短期运行成本最低，而是会放弃一部分收益较低、但会造成电池损耗的充放电行为。

其经济含义是：

```text
如果某次电池充放电带来的运行收益小于电池寿命损耗成本，则这次充放电不应执行。
```

因此，S4 的运行成本略有上升是合理的，因为它减少了一些原本用于降低短期成本的电池调节行为；但这些行为造成的寿命损耗更高，所以减少后综合成本更低。

---

### 4.2 EV 调度变化最明显：V2B 调用减少

EV 是本次对比中变化最明显的对象。

S3 中 EV 总吞吐量为：

```text
3349.736 kWh
```

S4 中 EV 总吞吐量为：

```text
3015.055 kWh
```

减少：

```text
334.682 kWh
```

其中 EV 放电量从：

```text
592.676 kWh
```

下降到：

```text
433.911 kWh
```

减少：

```text
158.765 kWh
```

这表明 S4 明显减少了 EV 的 V2B 放电行为。

原因在于：

```text
S3 只判断 EV 放电是否能降低运行成本；
S4 还要判断 EV 放电收益是否能覆盖 EV 电池寿命损耗。
```

在 S3 中，只要 EV 放电能够降低购电成本、削减峰值或改善系统平衡，模型就可能调用 EV。  
在 S4 中，EV 放电需要满足更严格的经济条件：

```text
EV 放电带来的运行成本节约 > EV 电池寿命损耗成本
```

因此，一些低收益的 V2B 放电被取消，EV 电池使用更谨慎。

这也符合实际场景：EV 电池属于车主资产，园区不应为了很小的短期电费收益频繁调用车主电池。

---

### 4.3 EV 充电量也下降，说明无效循环减少

除了放电量下降，EV 充电量也从：

```text
2757.060 kWh
```

下降到：

```text
2581.143 kWh
```

减少：

```text
175.917 kWh
```

这是因为 EV 的放电减少后，后续为了补回放电能量而产生的额外充电也会减少。

也就是说，S4 不仅减少了 EV 放电，还减少了：

```text
先放电、再补电
```

这种会增加电池吞吐量的循环行为。

因此，EV 总吞吐量下降并不是单纯减少充电服务，而是减少了低价值的充放电往返循环。

从服务质量看，S3 和 S4 的 EV 离站缺口均为：

```text
23.263 kWh
```

说明 S4 减少 EV 吞吐量并没有进一步恶化 EV 离站满足情况。该缺口主要来自前面已经识别的物理不可达 EV 会话，而不是寿命成本导致的额外不满足。

---

### 4.4 固定储能调度基本不变

固定储能吞吐量在 S3 和 S4 中均为：

```text
12952.421 kWh
```

固定储能寿命损耗成本也均为：

```text
712.383 元
```

说明在当前参数下，固定储能调度没有明显变化。

原因是固定储能单位吞吐寿命成本为：

```text
0.055 元/kWh
```

该成本相对较低，而固定储能在削峰填谷、光伏消纳和降低购电成本方面的收益仍然能够覆盖其寿命损耗成本。

因此，加入寿命损耗后，模型仍然认为固定储能值得使用。

这反映出固定储能和 EV 的角色差异：

```text
固定储能是园区资产，适合承担高频、计划性较强的能量调节；
EV 是用户资产，适合作为补充灵活性资源，不宜频繁参与 V2B。
```

---

### 4.5 外网购电行为变化较小

电网购电量从：

```text
64007.530 kWh
```

变为：

```text
63990.378 kWh
```

变化仅为：

```text
-17.152 kWh
```

峰值购电功率保持：

```text
607.550 kW
```

不变。

这说明，S4 减少 EV 放电后，并没有明显增加外网购电压力，也没有导致峰值购电功率上升。

原因可能是：

1. 固定储能仍维持原有调节作用；
2. 建筑柔性负荷仍参与削峰和转移；
3. EV 充电时段被重新安排，减少了低价值的充放电循环；
4. 系统通过多资源协同抵消了 EV V2B 减少带来的影响。

因此，S4 并不是简单地“少用 EV、多买电”，而是在保持系统运行约束的同时，减少不必要的电池吞吐。

---

### 4.6 建筑柔性负荷行为基本不变

建筑负荷转移量在 S3 和 S4 中均为：

```text
5649.376 kWh
```

建筑负荷削减量均为：

```text
693.408 kWh
```

说明引入电池寿命损耗后，建筑柔性负荷策略没有明显变化。

原因是建筑柔性负荷本身不产生电池寿命损耗，它的调度主要由舒适度惩罚、削减惩罚和峰值控制需求决定。

当前参数下，S4 主要通过调整 EV 充放电行为来降低寿命损耗，不需要额外改变建筑负荷调节方式。

---

### 4.7 光伏消纳率保持不变

S3 和 S4 的光伏消纳率均为：

```text
1.000
```

说明引入寿命损耗后，系统仍能实现光伏完全消纳。

这很重要，因为如果考虑寿命损耗导致大量弃光，则说明模型以牺牲新能源利用为代价减少电池损耗。当前结果没有出现这种情况。

也就是说，S4 在降低 EV 电池损耗的同时，没有降低光伏消纳效果。

## 6. 主要结论

第 3 小问可以形成以下结论：

1. **引入寿命损耗后，系统调度从“运行成本最小”转向“综合成本最小”。**  
   S4 运行成本比 S3 增加 `10.564 元`，但寿命损耗成本减少 `27.421 元`，综合成本降低 `16.857 元`。

2. **EV 调度变化最显著。**  
   EV 总吞吐量减少 `334.682 kWh`，EV 放电量减少 `158.765 kWh`，说明 S4 减少了收益不足以覆盖寿命损耗的 V2B 行为。

3. **EV 充电量同步下降，说明低价值充放电循环减少。**  
   由于 EV 放电减少，后续补电需求也减少，EV 充电量下降 `175.917 kWh`。

4. **固定储能调度基本不变。**  
   当前固定储能单位寿命成本较低，且其调节收益较高，因此仍然承担主要削峰填谷和光伏消纳任务。

5. **外网购电、峰值购电、建筑柔性负荷和光伏消纳基本稳定。**  
   说明 S4 减少 EV 电池损耗时，没有明显牺牲系统供能能力、光伏消纳效果或负荷服务质量。

6. **S4 更符合长期运行逻辑。**  
   模型只有在运行收益能够覆盖电池寿命损耗时才调用电池资源，避免为了短期电费收益过度消耗 EV 电池寿命。

## 7. 论文表达建议

论文中可以这样表述：

```text
为分析电池寿命损耗成本对系统调度行为的影响，本文将未考虑寿命损耗的 S3 完整协同方案与考虑寿命损耗的 S4 方案进行对比。两种方案采用相同的系统约束，差异仅在于 S4 目标函数中加入了固定储能与 EV 电池寿命损耗成本。结果表明，引入寿命损耗后，系统调度行为的主要变化集中在 EV 侧。S4 中 EV 总吞吐量由 3349.736 kWh 降至 3015.055 kWh，EV 放电量由 592.676 kWh 降至 433.911 kWh，说明模型减少了收益不足以覆盖电池损耗的 V2B 放电行为。与此同时，固定储能吞吐量、峰值购电功率、建筑柔性负荷调整量和光伏消纳率基本保持不变，表明固定储能在当前参数下仍具有较高调节价值。虽然 S4 的运行成本较 S3 增加 10.564 元，但寿命损耗成本降低 27.421 元，使综合成本下降 16.857 元。因此，引入电池寿命损耗成本后，系统由单纯追求短期运行成本最低，转向兼顾长期电池健康的综合成本最优调度。
```

本小问核心结论可以概括为：

```text
引入电池寿命损耗后，系统不会盲目频繁调用电池资源，而是只在运行收益能够覆盖寿命损耗时才进行充放电；其中 EV V2B 调用减少最明显，固定储能仍保持主要调节作用。
```
