# 第三问第2小问：碳交易价格对调度决策的影响

## 1. 分析目标

在**无碳上限**（纯碳税模式）下，分析碳价水平对购电、储能充放电、EV V2B 决策的影响。

## 2. 计算方法

```
1. 固定 cap=None（无硬约束），碳价通过目标函数中的购电碳成本加成起作用
2. 对碳价 0/100/300/500 元/吨，分别求解 S5
3. 记录：碳排放、运行成本、碳交易成本、综合成本、购电量、储能吞吐、EV 放电量
4. 对比各碳价下调度行为变化
```

碳价进入目标函数的方式：

```
c[grid_buy[t]] += carbon_price × grid_carbon_kg_per_kwh[t] × Δt
```

碳交易成本报表：`carbon_price × 全排放`（碳税模式，免费配额 = 0）。

## 3. 碳价场景

| 场景 | 碳价（元/吨） | 折合元/kg | 参考碳市场 |
|------|------:|------:|------|
| 无碳成本 | 0 | 0 | 基准对照 |
| 低碳价 | 100 | 0.10 | 中国全国碳市场（60–120） |
| 中碳价 | 300 | 0.30 | 中间情景 |
| 高碳价 | 500 | 0.50 | EU ETS（450–700） |

碳价对等值电价的影响（以 500 元/吨为例）：

- 谷时：电网电价 0.42 + 碳成本 0.50×0.55 = 0.42 + 0.275 = **0.695 元/kWh（+65%）**
- 峰时：电网电价 0.75 + 碳成本 0.50×0.74 = 0.75 + 0.37 = **1.12 元/kWh（+49%）**

## 4. 关键结果

| 碳价 元/吨 | 排放 kg | 运行成本 元 | 碳交易成本 元 | 综合成本 元 | EV 放电 kWh |
|------:|------:|------:|------:|------:|------:|
| 0 | 42,237 | 44,485 | 0 | 45,458 | 434 |
| 100 | 42,126 | 44,486 | 4,213 | 49,671 | 434 |
| 300 | 42,080 | 44,495 | 12,624 | 58,093 | 434 |
| 500 | 41,651 | 44,649 | 20,825 | 66,445 | 434 |

## 5. 结果分析

### 5.1 碳价减排效果极弱

碳价从 0 → 500 元/吨，碳排放仅从 42,237 → 41,651 kg（**−1.4%**）。

尽管 500 元/吨碳价使等值电价上涨了约 50–65%，但在这种极端价格信号下减排仍然微乎其微。原因是：

- 园区刚性负荷占比高（办公室、实验室、教学楼），无法大幅削减
- 光伏容量已充分利用（消纳率 100%）
- 固定储能仅 1,200 kWh，EV V2B 吞吐量仅 ~434 kWh——灵活资源容量有限
- 碳强度的时段差异（0.55–0.74 kg/kWh，最大波动 35%）不足以支撑大幅碳强度套利

### 5.2 核心结论

**纯碳价信号对该园区基本无效。必须依靠碳配额硬约束才能实现有意义的减排。**

### 5.3 EV V2B 完全不受影响

各碳价下 EV 放电量恒定为 434 kWh。EV 的单位寿命损耗成本远高于碳价带来的边际收益。

### 5.4 碳价主要影响成本构成

- 碳价 0 → 500 元/吨时，碳交易成本占总成本从 0% → 31%
- 对园区财务影响显著，但对实际调度行为影响有限

## 6. 与前一小问的关联

前一小问（严格 cap = 93% 自然排放）实现了减排 7%，碳价信号（500 元/吨）仅实现 1.4%。这两个数字直接对比说明：**碳排放上限的约束力远强于碳价信号**。

更深层的原因将在第3小问通过影子价格分析揭示——园区的边际减排成本（~917 元/吨）远超测试碳价（0–500 元/吨），因此价格信号无法驱动减排。
