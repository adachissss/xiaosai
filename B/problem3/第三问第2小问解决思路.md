# 第三问第 2 小问解决思路：不同碳交易价格对系统调度的影响

## 1. 小问目标

第三问第 2 小问要求：

```text
分析碳交易价格水平如何影响园区从外部购电、储能充放电及电动车 V2B 决策。
```

与第 1 小问不同，本小问不设碳排放硬上限（cap = None），仅通过碳交易价格信号（加入目标函数）影响系统决策。这模拟的是纯粹的碳市场机制——排放越多，碳交易成本越高，但排放量本身不受硬性限制。

---

## 2. 模型基础：cap-and-trade（碳配额交易）

S5 采用 cap-and-trade 机制。目标函数中碳交易成本被直接加入：

```text
min 运行成本 + 固定储能寿命损耗 + EV 寿命损耗 + 碳交易成本
```

碳交易成本为：

```text
C_carbon = carbon_price × (Total_Carbon - free_allowance)
```

其中 free_allowance 是免费配额（常量），不影响优化。在 LP 目标函数中：

```python
c[grid_buy[t]] += carbon_price × carbon_intensity[t] × Δt
```

碳交易成本事后核算：

```python
carbon_trading_cost = carbon_price × (total_emissions - free_allowance)  # 可为负（收入）
```

碳价越高，高碳强度时段的购电成本相对上升越多，LP 倾向于在高碳时段减少购电。当排放低于配额时，园区可获得碳交易收入。

---

## 3. 碳价情景设计

固定免费配额为 S4 基线排放（42237 kg），变化碳价：

| 场景 | carbon_price（元/kg） | carbon_price（元/吨） | 含义 |
|---:|---:|---:|---|
| 无碳成本 | 0.00 | 0 | 基线——碳价为零，排放=配额，交易成本=0 |
| 低碳价 | 0.10 | 100 | 参照当前国内碳市场水平（约 60-120 元/吨） |
| 中碳价 | 0.30 | 300 | 模拟碳市场较高水平 |
| 高碳价 | 0.50 | 500 | 模拟碳市场极端水平或附加惩罚 |

关键变化：碳交易成本 = 碳价 × (实际排放 - 42237)。当排放低于配额时，碳交易成本为负（卖配额收入）。

---

## 4. 关键代码说明

### 4.1 碳交易成本加入目标函数

在 `carbon_aware_strategy.py:117-119`：

```python
carbon_intensity = ts["grid_carbon_kg_per_kwh"].to_numpy()
c[grid_buy] += carbon_price_cny_per_kg * carbon_intensity * DT
```

含义：每个时段 `grid_buy[t]` 的目标系数增加 `碳价 × 碳强度 × 步长`。这等价于对购电征收与碳强度成正比的"碳税"。在 LP 看来，不同的 carbon_price 会改变购电在不同时段的相对成本，从而影响购电时段的分布决策。

### 4.2 场景遍历

在 `solve_problem3.py:282-301`：

```python
for label, price in price_scenarios:
    sched, ev_r, info = build_and_solve_carbon_aware(
        data, carbon_price_cny_per_kg=price, carbon_cap_kg=None, scheme_name=f"S5_price_{label}"
    )
    m = extract_metrics(sched, ev_r, data, data_dir, price, None)
    m["scenario_label"] = label
    metrics_list.append(m)
```

含义：对每个碳价调用 S5 求解器，不设碳排放上限。提取完整的指标（碳排放、运行成本、碳交易成本、储能吞吐量、EV 放电等）进行对比。

### 4.3 碳交易成本计算

在 `solve_problem3.py:87`：

```python
carbon_trading_cost = total_carbon_kg * carbon_price
```

含义：碳交易成本 = 实际碳排放 × 碳价。在 S5 中，这部分成本已被纳入目标函数，因此 LP 求解时会自动权衡减排和碳税成本的关系。

## 6. 结果分析

### 6.1 碳价对排放的抑制效果有限

即使碳价升至 500 元/吨，碳排放仅从 42237 kg 降到 41651 kg（-1.4%）。原因：

1. **园区可调资源接近饱和**：储能吞吐量已达 12952 kWh，光伏消纳率 100%，没有更多"无碳资源"
2. **碳强度波动范围有限**：0.55-0.74 kg/kWh，最高仅比最低高 35%
3. **负荷刚性较强**：办公、实验室、教学中心的基础需求无法大幅削减

### 6.2 碳价提高使园区从碳交易获利（而非受损）

与旧模型（碳税制）的关键区别：旧模型下碳价 500 时园区需支付 20825 元碳税；新模型下园区反而赚 293 元。原因：

- 免费配额（42237 kg）覆盖了大部分排放
- 碳价驱动减排（排放降至 41651 kg），剩余配额可卖出获利
- 碳价越高，卖出剩余配额的单位收益越高

### 6.3 综合成本随碳价上升而下降（反直觉）

| 碳价 | 综合成本 | 变化 |
|---:|---:|---:|
| 0 | 45458 | 基线 |
| 100 | 45448 | -10 |
| 300 | 45421 | -37 |
| 500 | 45326 | -132 |

这是一个重要发现：碳价上升后，运行成本略有上升（+164 元），但碳交易收入上升更多（+293 元），导致**综合成本不升反降**。这说明在 cap-and-trade 下，碳价对园区可能不是负担，而是收入来源。

### 6.4 EV 放电完全不变

所有碳价下 EV 放电均为 433.9 kWh。V2B 的寿命损耗成本是主要制约因素，碳价信号不足以改变 V2B 的边际收益。

### 6.5 购电调整

| 碳价 | 购电总量 kWh | 购电峰值 kW |
|---:|---:|---:|
| 0 | 63990 | 608 |
| 100 | 63986 | 606 |
| 300 | 63973 | 598 |
| 500 | 63527 | 580 |

峰值购电下降（-4.6%）比总量下降（-0.7%）更明显，说明碳价主要通过调整峰值时段购电来减排。

## 8. 第 1 小问与第 2 小问的结果对比

cap-and-trade 机制下两个参数的不同角色：

| 维度 | 免费配额（第 1 小问） | 碳交易价格（第 2 小问） |
|---:|---|
| 角色 | 财务分配工具 | 行为调节工具 |
| 影响调度？ | 否——常量，不参与优化 | 是——改变购电有效成本 |
| 影响财务？ | 是——决定园区是买家还是卖家 | 是——影响单位碳交易的收益/成本 |
| 政策意义 | 配额分配决定行业公平性 | 碳价设定决定减排力度 |
| 配合使用 | 合理碳价 + 适度配额 = 最优 | |

## 9. 本小问结论

第三问第 2 小问已完成：

```text
在 cap-and-trade 机制下，固定免费配额（基线排放 42237 kg），通过不同碳价
（0/100/300/500 元/吨）分析了价格信号对调度和财务的影响。
```

核心发现：
1. 碳价对减排的边际效果有限——500 元/吨仅减排 1.4%
2. **碳价上升反而降低了综合成本**（碳交易收入超过运行成本上升）
3. EV V2B 放电量在所有碳价下不变——寿命成本是 V2B 的主要制约因素
4. 碳价主要通过抑制峰值购电来减排（峰值 -4.6% vs 总量 -0.7%）
5. cap-and-trade 下碳价对园区可能是收入来源而非负担
