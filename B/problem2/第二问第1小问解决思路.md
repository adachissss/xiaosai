# 第二问第 1 小问解决思路：电池寿命损耗指标与成本折算

## 1. 小问目标

第二问第 1 小问要求：

```text
提出一种可以衡量固定储能与电动车电池寿命损耗的指标或成本折算方式。
```

也就是说，这一小问不需要重新优化调度，而是要回答：

1. 固定储能和 EV 电池的寿命损耗应该用什么指标衡量；
2. 如何把这个寿命损耗折算成经济成本；
3. 这个指标后续如何加入第二问第 2 小问的优化目标函数。

当前采用的解决思想是：

```text
用电池等效吞吐量衡量寿命损耗强度，
再用单位吞吐寿命成本把寿命损耗折算成经济成本。
```

---

## 2. 为什么选择等效吞吐量模型

电池寿命损耗主要来自充放电过程。

如果只看电池最终电量变化，会低估电池实际经历的损耗。例如：

```text
上午充电 100 kWh，下午放电 100 kWh
```

最终电量变化可能为 0，但电池实际经历了：

```text
100 + 100 = 200 kWh
```

的能量流动。

因此，电池寿命损耗不应使用净电量变化，而应使用累计充放电能量，即：

```text
等效吞吐量 = 充电电量 + 放电电量
```

该模型的优点是：

1. 与题目给定的充放电功率数据直接匹配；
2. 形式简单，容易解释；
3. 仍然是线性的，可以直接加入线性规划目标函数；
4. 同时适用于固定储能和 EV 电池；
5. 适合数学建模论文表达。

---

## 3. 电池等效吞吐量定义

附件数据时间分辨率为 15 分钟，因此：

```text
Δt = 0.25 h
```

若某个时段的充电功率为 `P_charge[t]`，放电功率为 `P_discharge[t]`，则该时段的电池等效吞吐量为：

```text
(P_charge[t] + P_discharge[t]) × Δt
```

整个调度周期内的电池等效吞吐量为：

```text
E_throughput = Σ_t (P_charge[t] + P_discharge[t]) × Δt
```

单位为：

```text
kWh
```

含义：

```text
电池在调度周期内累计经历了多少充放电能量流动。
```

---

## 4. 固定储能寿命损耗计算

固定储能的等效吞吐量定义为：

```text
E_bat_throughput
= Σ_t (P_bat_ch[t] + P_bat_dis[t]) × Δt
```

其中：

- `P_bat_ch[t]`：第 `t` 时段固定储能充电功率；
- `P_bat_dis[t]`：第 `t` 时段固定储能放电功率；
- `Δt = 0.25 h`。

固定储能寿命损耗成本定义为：

```text
C_bat_deg = c_bat_deg × E_bat_throughput
```

其中 `c_bat_deg` 为固定储能单位吞吐寿命损耗成本。

本题中该参数不是人为假设，而是直接来自附件：

```text
B/B_data/asset_parameters.csv
```

字段为：

```text
stationary_battery_degradation_cost_cny_per_kwh_throughput
```

数值为：

```text
c_bat_deg = 0.055 元/kWh
```

因此固定储能寿命损耗成本为：

```text
C_bat_deg
= 0.055 × Σ_t (P_bat_ch[t] + P_bat_dis[t]) × 0.25
```

---

## 5. EV 电池寿命损耗计算

EV 电池与固定储能类似，也使用等效吞吐量。

对第 `i` 辆 EV：

```text
E_ev_throughput[i]
= Σ_t (P_ev_ch[i,t] + P_ev_dis[i,t]) × Δt
```

其中：

- `P_ev_ch[i,t]`：第 `i` 辆 EV 在第 `t` 时段的充电功率；
- `P_ev_dis[i,t]`：第 `i` 辆 EV 在第 `t` 时段的放电功率；
- `Δt = 0.25 h`。

第 `i` 辆 EV 的寿命损耗成本为：

```text
C_ev_deg[i]
= c_ev_deg[i] × E_ev_throughput[i]
```

全部 EV 的寿命损耗成本为：

```text
C_ev_deg
= Σ_i c_ev_deg[i] × Σ_t (P_ev_ch[i,t] + P_ev_dis[i,t]) × Δt
```

EV 的单位吞吐寿命成本来自：

```text
B/B_data/ev_sessions.csv
```

字段为：

```text
degradation_cost_cny_per_kwh_throughput
```

不同 EV 会话可以有不同的单位损耗成本。

---

## 6. 总寿命损耗成本

固定储能和 EV 的寿命损耗成本可以统一加总，得到系统总寿命损耗成本：

```text
C_total_deg = C_bat_deg + C_ev_deg
```

展开为：

```text
C_total_deg
= c_bat_deg × Σ_t (P_bat_ch[t] + P_bat_dis[t]) × Δt
+ Σ_i c_ev_deg[i] × Σ_t (P_ev_ch[i,t] + P_ev_dis[i,t]) × Δt
```

这就是第二问第 1 小问提出的核心成本折算方式。

后续第二问第 2 小问可以直接把它加入目标函数：

```text
min 运行成本 + C_total_deg
```

## 8. 关键代码说明

### 8.1 固定储能寿命损耗计算代码

关键函数：

```python
def battery_degradation_cost(
    schedule: pd.DataFrame,
    unit_cost_cny_per_kwh: float = DEFAULT_STATIONARY_BATTERY_DEG_COST,
) -> tuple[float, float, float, float]:
    charge_kwh = float((schedule["battery_charge_kw"] * DT).sum())
    discharge_kwh = float((schedule["battery_discharge_kw"] * DT).sum())
    throughput_kwh = charge_kwh + discharge_kwh
    cost_cny = throughput_kwh * unit_cost_cny_per_kwh
    return charge_kwh, discharge_kwh, throughput_kwh, cost_cny
```

含义：

1. `battery_charge_kw × DT` 得到固定储能充电电量；
2. `battery_discharge_kw × DT` 得到固定储能放电电量；
3. 两者相加得到固定储能等效吞吐量；
4. 乘以 `unit_cost_cny_per_kwh` 得到固定储能寿命损耗成本。

### 8.2 读取固定储能寿命成本参数代码

关键函数：

```python
def load_stationary_battery_degradation_cost(data_dir: Path) -> float:
    asset = pd.read_csv(data_dir / "asset_parameters.csv")
    key = "stationary_battery_degradation_cost_cny_per_kwh_throughput"
    matched = asset.loc[asset["parameter"] == key, "value"]
    if matched.empty:
        return DEFAULT_STATIONARY_BATTERY_DEG_COST
    return float(matched.iloc[0])
```

含义：

- 优先从 `asset_parameters.csv` 中读取固定储能寿命成本；
- 当前读到的值为 `0.055 元/kWh`；
- 如果字段缺失，才使用默认兜底值。

### 8.3 EV 寿命损耗计算代码

关键函数：

```python
def ev_degradation_cost(
    schedule: pd.DataFrame,
    ev_result: pd.DataFrame,
    data_dir: Path,
) -> tuple[float, float, float, float]:
    charge_kwh = float((schedule["ev_charge_total_kw"] * DT).sum())
    discharge_kwh = float((schedule["ev_discharge_total_kw"] * DT).sum())
    throughput_kwh = charge_kwh + discharge_kwh

    ev_sessions = pd.read_csv(data_dir / "ev_sessions.csv")
    cost_col = "degradation_cost_cny_per_kwh_throughput"
    cost_map = ev_sessions.set_index("session_id")[cost_col]

    ev = ev_result.copy()
    ev["net_energy_gain_kwh"] = (
        ev["final_energy_kwh"] + ev.get("shortfall_kwh", 0.0)
    ) - ev["initial_energy_kwh"]
    ev["estimated_throughput_kwh"] = np.maximum(0.0, ev["net_energy_gain_kwh"])
    ev["unit_degradation_cost_cny_per_kwh"] = ev["session_id"].map(cost_map)

    estimated_cost_cny = float(
        (ev["estimated_throughput_kwh"] * ev["unit_degradation_cost_cny_per_kwh"]).sum()
    )
    avg_unit_cost = estimated_cost_cny / float(ev["estimated_throughput_kwh"].sum())
    cost_cny = throughput_kwh * avg_unit_cost
    return charge_kwh, discharge_kwh, throughput_kwh, cost_cny
```

说明：

第一问结果中只保存了 EV 聚合充放电功率，没有保存每辆 EV 在每个时段的充放电功率。因此第 1 小问中先用 EV 聚合功率计算总吞吐量，再用 EV 会话结果估算加权平均 EV 单位寿命成本。

### 8.4 EV 加权平均单位寿命成本

由于每辆 EV 的单位寿命损耗成本不同，不能简单使用所有 EV 寿命成本参数的普通平均值。当前采用的是按估计补电需求加权的平均单位寿命成本。

理论上，车辆级精确公式应为：

```text
C_ev_deg = Σ_i c_ev_deg[i] × E_ev_throughput[i]
```

其中权重应当是每辆 EV 的真实吞吐量 `E_ev_throughput[i]`。

但是第一问输出结果没有保存车辆级逐时段充放电功率，因此第 1 小问暂时用每辆车的估计补电需求作为权重：

```text
estimated_throughput_i
= max(0, final_energy_i + shortfall_i - initial_energy_i)
```

加权平均单位寿命成本为：

```text
c_ev_avg
= Σ_i(estimated_throughput_i × c_ev_deg[i]) / Σ_i estimated_throughput_i
```

然后用聚合 EV 总吞吐量计算 EV 总寿命损耗成本：

```text
C_ev_deg = E_ev_total_throughput × c_ev_avg
```

这样处理的含义是：

- 需要补电越多的 EV，在平均寿命成本中权重越大；
- 单位寿命成本较高且补电需求较大的 EV，会对 EV 总寿命损耗成本产生更大影响；
- 这是一种基于第一问聚合结果的近似计算。

后续第 2 小问重新求解 LP 时，会保留车辆级变量 `P_ev_ch[i,t]` 和 `P_ev_dis[i,t]`，届时可以更精确地计算：

```text
Σ_i c_ev_deg[i] × Σ_t (P_ev_ch[i,t] + P_ev_dis[i,t]) × Δt
```

因此，第 1 小问中的加权平均方法主要用于解释和试算；第 2 小问中的 S4 模型会使用车辆级精确寿命成本项。

## 10. 当前数值结果

基于第一问四个方案，得到如下寿命损耗成本：

| 方案 | 固定储能吞吐量/kWh | 固定储能损耗成本/元 | EV 吞吐量/kWh | EV 损耗成本/元 | 总寿命损耗成本/元 |
|---|---:|---:|---:|---:|---:|
| S0 朴素方案 | 0.000 | 0.000 | 2100.355 | 181.105 | 181.105 |
| S1 规则储能 | 456.000 | 25.080 | 2100.355 | 181.105 | 206.185 |
| S2 部分协同 | 12962.224 | 712.922 | 2100.355 | 180.736 | 893.659 |
| S3 完整协同 | 12952.421 | 712.383 | 3349.736 | 288.246 | 1000.629 |

可以看出：

1. S0 固定储能不动作，因此固定储能寿命损耗为 0；
2. S1 只使用规则储能，固定储能吞吐量较小；
3. S2 大量调用固定储能，因此固定储能寿命损耗明显增加；
4. S3 在 S2 基础上允许 EV V2B，因此 EV 吞吐量和 EV 寿命损耗进一步增加；
5. 如果只看寿命损耗，S3 最大；但如果叠加第一问运行成本，S3 的综合成本仍低于 S2。

## 12. 第 1 小问论文表达建议

论文中可以这样表述：

```text
为衡量固定储能与电动车电池因频繁充放电产生的寿命损耗，本文采用等效充放电吞吐量作为寿命损耗的物理指标。等效吞吐量定义为调度周期内电池充电电量与放电电量之和。进一步结合附件中给定的单位吞吐寿命损耗成本，将电池寿命损耗折算为经济成本。该方法既能反映电池使用强度，又保持线性形式，便于在后续优化模型中直接加入目标函数。
```

对应公式为：

```text
E_throughput = Σ(P_charge + P_discharge) × Δt
```

```text
C_total_deg
= c_bat_deg × Σ_t(P_bat_ch[t] + P_bat_dis[t]) × Δt
+ Σ_i c_ev_deg[i] × Σ_t(P_ev_ch[i,t] + P_ev_dis[i,t]) × Δt
```

第二问第 1 小问已经形成完整解决方案：

```text
以等效充放电吞吐量作为电池寿命损耗物理指标，
以单位吞吐寿命成本将其折算为经济成本，

```
